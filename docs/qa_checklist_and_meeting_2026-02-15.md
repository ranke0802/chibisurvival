# QA-개발 합동 점검 회의록 및 체크리스트

- 일시: 2026-02-15
- 목적: 게임 기능 및 UX/UI 정상 출력 여부를 실제 화면 구동으로 확인
- 환경: macOS, Vite Dev Server(`http://127.0.0.1:4173`), Playwright Chromium Headless

## 2) QA-개발 사전 회의 결론
1. 기능 정상성 확인은 "화면 전이 + 입력 + 모달 + HUD" 최소 경로를 우선 검증한다.
2. UX/UI 정상 출력 확인은 "텍스트/버튼/HUD/캔버스" 가시성과 "오류 로그 없음"을 필수 조건으로 둔다.
3. 자동화 재실행 가능성을 위해 Playwright 스모크 스크립트를 프로젝트에 보관한다.

## 3) 확정 체크리스트
1. 앱 초기 진입 성공
2. 타이틀 화면 텍스트 정상 노출
3. "게임 시작" 버튼 클릭 가능
4. 캐릭터 선택 화면 정상 전환
5. "출전" 버튼 클릭 가능
6. 인게임 캔버스 정상 렌더
7. 스테이지 HUD 정상 노출
8. EXP HUD 정상 노출
9. ESC 입력으로 일시정지 가능
10. 일시정지 모달 정상 노출
11. "계속하기"로 게임 복귀 가능
12. 콘솔 에러 없음
13. 런타임 예외 없음
