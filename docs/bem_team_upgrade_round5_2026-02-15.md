# bem 팀 고도화 회의 및 실행 결과 (Round 5)

- 일시: 2026-02-15
- 참석: QA, 개발, 게임디자인, 그래픽디자인, 사운드디자인
- 목표: 보스 패턴 인지성 강화 + 전사/궁수 공격 모션 체감 개선

## 1) 회의 결론
1. 게임디자인
- 보스 텔레그래프가 동시에 여러 개일 때 "무슨 패턴이 먼저 터지는지"를 UI로 알려줘야 회피 판단이 빨라짐.

2. 그래픽디자인
- 캐릭터 이동은 부드러워졌지만 공격 타이밍이 몸동작에서 더 분명히 보여야 타격감이 산다.

3. 사운드디자인
- 패턴별 예고음을 이미 넣었으므로, UI 경고도 패턴별로 맞춰주면 시청각 일관성이 좋아진다.

4. QA
- 신규 기능은 빌드 통과와 함께 다음을 확인:
  - 패턴 경고 배지가 상황에 맞게 바뀌는지
  - 전사/궁수 공격 순간 반동/기울기 변화가 보이는지
  - 기존 HUD 및 설정 기능 회귀 없는지

## 2) 반영한 고도화
1. 보스 패턴 경고 배지 UI 추가
- 현재 보스 텔레그래프 중 가장 먼저 발동할 패턴을 실시간 표기:
  - 원형 폭발(`circle`)
  - 직선 관통(`line`)
  - 부채 베기(`cone`)
- 아이콘 + 라벨 + 남은 시간(초) 표시
- 반영 파일: `src/App.tsx`, `src/styles.css`

2. 플레이어 공격 모션 펄스 시스템 추가
- 엔진에서 기본 공격 발동 시 `playerAttackPulse` 값을 올리고 시간 경과로 감쇠.
- 렌더러에서 캐릭터별(전사/궁수/마법사) 반동/기울기/상하 움직임에 반영.
- 반영 파일: `src/game/types.ts`, `src/game/engine.ts`, `src/game/renderer.ts`

3. 공격 실패 시 딜레이 보정
- 공격 대상이 없으면 긴 쿨다운 대신 짧은 재시도 딜레이를 주어 입력 체감 개선.
- 반영 파일: `src/game/engine.ts`

## 3) 검증
1. 빌드 검증
- 명령: `npm run build`
- 결과: 성공

2. QA 체크 포인트
- 보스전에서 패턴 배지 색상/라벨이 텔레그래프 종류와 일치하는지
- 전사/궁수 공격 시 기존 대비 반동 연출이 강화되었는지
- 설정 모달 프리셋/고대비 옵션이 정상 동작하는지
