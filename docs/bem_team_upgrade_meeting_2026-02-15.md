# bem 팀 심화 고도화 회의록

- 일시: 2026-02-15
- 팀명: bem 팀
- 참석: QA 담당자, 개발자, 게임 디자이너, 그래픽 디자이너, 사운드 디자이너
- 목표: 현재 프로젝트를 상용 수준에 가깝게 고도화하고, 즉시 플레이 체감 개선을 반영

## 1) 직군별 진단
1. QA 담당자
- 문제: 장시간 플레이 시 정보 밀집과 시각 피로 누적.
- 요구: 옵션 기반으로 가시성/이펙트 강도/사운드 강도 조절 가능해야 함.

2. 개발자
- 문제: 보스전 패턴 예고가 약해 피격이 불공정하게 느껴질 수 있음.
- 요구: 보스 위험 구간 텔레그래프를 명시하고, 난이도 상승에 따라 패턴이 자연스럽게 강화되어야 함.

3. 게임 디자이너
- 문제: 보스전 긴장 곡선이 평면적임.
- 요구: "경고 -> 회피 -> 폭발" 리듬을 추가해 보스전 의사결정 재미를 강화.

4. 그래픽 디자이너
- 문제: 현재 이펙트는 강하지만 사용자 취향 대응이 부족.
- 요구: 미니맵/데미지 텍스트/이펙트 출력량을 토글 가능하게 하여 가독성을 개인화.

5. 사운드 디자이너
- 문제: 사용자 환경(이어폰/스피커)에 따른 볼륨 최적점이 다름.
- 요구: 마스터/BGM/SFX 분리 볼륨을 실시간 조절 가능하게 제공.

## 2) 회의 결론(우선순위)
1. 보스 텔레그래프 패턴 도입(게임성 핵심)
2. 실시간 설정 모달 도입(UX 개인화)
3. 카메라 흔들림/사운드 강도 슬라이더 제공(피로도 관리)
4. 렌더러 옵션화(데미지 텍스트, 미니맵, FX 출력 제어)

## 3) 실행 결과(코드 반영 완료)
1. 보스 텔레그래프 공격 시스템 추가
- 보스가 주기적으로 플레이어 위치에 위험지대를 예고하고, 카운트다운 후 폭발 피해를 부여.
- 반영 파일: `src/game/types.ts`, `src/game/engine.ts`, `src/game/renderer.ts`

2. 실시간 설정 모달 추가
- 이펙트 간소화, 데미지 텍스트 표시, 미니맵 표시 토글 제공.
- 카메라 흔들림 강도 슬라이더 제공.
- 반영 파일: `src/App.tsx`, `src/styles.css`

3. 오디오 믹스 제어 추가
- 마스터/BGM/SFX 볼륨을 실시간 조절 가능하도록 `AudioManager` 확장.
- 반영 파일: `src/audio/AudioManager.ts`, `src/App.tsx`

4. 보스 fallback 스프라이트 경로 정리
- 누락 가능 경로를 안전한 보스 에셋 경로로 교체.
- 반영 파일: `src/game/renderer.ts`

## 4) 검증 결과
- 빌드 검증: `npm run build` 통과
- 상태: 이번 회의에서 합의된 핵심 고도화 항목 1차 반영 완료
