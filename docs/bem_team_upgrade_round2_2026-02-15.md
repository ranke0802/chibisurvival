# bem 팀 고도화 회의 및 실행 결과 (Round 2)

- 일시: 2026-02-15
- 참석: QA, 개발, 게임디자인, 그래픽디자인, 사운드디자인
- 목적: 플레이 안정성/가독성/난이도 표현/개인화 설정 강화

## 회의 핵심 결론
1. 보스전은 패턴 다양성이 필요하다.
- 원형 장판만으로는 회피 패턴이 단조로움.
- 직선 관통형 텔레그래프를 추가해 리듬을 강화한다.

2. 설정은 세션 종료 후 유지되어야 한다.
- 사용자마다 카메라 흔들림/볼륨/이펙트 선호가 다르므로 localStorage로 저장한다.

3. 렌더 옵션은 저사양/모바일 가독성에 직접 영향이 크다.
- 데미지 텍스트, 미니맵, FX를 개별 토글 가능 상태를 유지한다.

## 실행한 고도화 항목
1. 보스 텔레그래프 패턴 다변화
- `circle` 패턴 + `line` 패턴 추가
- line 패턴은 보스 위치 기준 플레이어 방향으로 관통 예고 후 폭발 피해
- 반영 파일:
  - `src/game/types.ts`
  - `src/game/engine.ts`
  - `src/game/renderer.ts`

2. 설정 영속화
- 실시간 설정을 `localStorage`에 저장/복원
- 저장 키: `bem_live_settings_v1`
- 반영 파일:
  - `src/App.tsx`

3. 유지보수 정리
- 보스 fallback 스프라이트 경로는 안전한 파일로 유지
- 렌더 옵션 연동 유지(FX/데미지텍스트/미니맵)

## 기대 효과
1. 보스전 체감 난이도와 회피 재미 상승
2. 사용자 맞춤형 시각/청각 환경 유지
3. 다음 세션 재진입 시 설정 재조정 시간 절감
